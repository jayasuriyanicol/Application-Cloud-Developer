# Esercizio 1 – Operazioni con variabili

Crea una classe chiamata `OperazioniVariabili` che:

- Definisca due variabili intere (ad esempio `a = 20` e `b = 4`)
- Stampi:
  - Il valore delle variabili
  - La somma
  - La differenza
  - Il prodotto
  - Il risultato della divisione

**Obiettivo:** usare variabili invece di numeri scritti direttamente.

# Esercizio 2 – Messaggio personalizzato

Crea una classe chiamata `SalutoPersonalizzato` che:

- Definisca due variabili di tipo `String`: `nome` e `città`
- Stampi una frase del tipo:

'Ciao, mi chiamo [nome] e vengo da [città]'
