# Application Cloud Developer ☁️

Welcome to the **Application Cloud Developer** course !

This training program provides me *both theoretical knowledge and hands-on experience* in the environment of cloud application development.  
It is mainly focused on coding and on the creation and organization of **OPP, Application and Databases.**

# 🗂 Repository Structure  

- **Subjects & Organization**: Folders are organized by academic subject, covering the core modules of FULLSTACK & CLOUD program.

- **Learning Progression**: Within each subject folder, exercises are structured chronologically, following the lesson sequence (generally numbered in ascending order) to reflect the learning path.

- **Hands-on Practice**: Each module focuses on practical coding exercises designed to reinforce theoretical concepts through real-world implementation.

- **Internship (Stage)**: A dedicated section for the internship phase is planned. This area is currently under development, awaiting final details and structural guidelines before the official start.


# ✅ Key Features  

- **Hands-on** practice mostly coding in **Python** and **Java**
- **OPP** and **Developing** approach with **Microframework/Framework**
